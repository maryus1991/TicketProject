# for run the script
 
install python

then install virtualenv 
```bash
pip install virtualenv
```

then create virtual environment
```bash
virtualenv venv
```


then activate it by 
```bash
./venv/bin/activate
```

then install the pachages
```bash
pip install -r requirement.txt
```

then migarate
```bash
python manage.py migrate
```

then run the server
```bash
python manage.py runserver
```
